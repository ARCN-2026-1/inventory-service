# 1. Propósito del Sistema

El sistema permite gestionar la reserva de habitaciones de hotel garantizando:

- Disponibilidad en un rango de fechas sin sobreventa
- Búsqueda y filtrado de habitaciones por tipo de habitación y precio
- Aplicación de reglas de negocio: cancelaciones, confirmaciones y pagos
- Consistencia e integridad de datos entre todos los microservicios
